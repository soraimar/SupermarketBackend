#Shopping-car

CRUD de carrito de compras implementado en tienda Walmart

#Technologies
* [MongoDB](https://github.com/walmartdigital/brand-discounts-db)
* [Nestjs](https://nestjs.com/)

# Installation
1. Install node.js
```bash
$ npm install
```
2. Install [nest.js](https://docs.nestjs.com/)
```bash
$ npm i -g @nestjs/cli
```

# Running the app
```bash
$ npm run start
```
